# RFCs

本目录存放 Sleepless 的功能与设计 RFC（Request for Comments）。

## 命名与流程

- 文件命名：`NNN-short-name.md`，如 `001-caffeine-parity.md`。
- 新 RFC 需包含：Summary、Context/Problem、Goals、Proposed Solution、Implementation Details、Alternatives、Risks、Testing。
- 实现前需经过评审与确认，参见项目根目录下 RFC 相关规范或 `.cursor` 技能。
